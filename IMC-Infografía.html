El Problema

Tradicionalmente, el Índice de Masa Corporal (IMC) se calcula con una fórmula simple: IMC = peso / altura². Sin embargo, en la era de la IA, surge la oportunidad de explorar enfoques innovadores. El objetivo de este estudio es comparar la simplicidad y transparencia del método tradicional frente a la capacidad de aproximación y aprendizaje de una Red Neuronal Artificial.

Dos Filosofías Computacionales

Enfoque Tradicional

Programación Basada en Reglas

El programador conoce la regla y la codifica explícitamente.

  • Lógica Imperativa: Se dan instrucciones paso a paso.
  • Determinista: Misma entrada, siempre la misma salida.
  • Limitación: No aprende, requiere conocimiento previo absoluto.

Inteligencia Artificial

Machine Learning (TensorFlow)

El sistema infiere las reglas a partir de datos de entrada y salida.

  • Aprendizaje: Ajusta Pesos (W) y Sesgos (b).
  • Generalización: Capaz de encontrar patrones complejos.
  • Caja Negra: Menor interpretabilidad directa.

Arquitectura del Modelo

Se utilizó TensorFlow para crear un modelo de regresión lineal. El proceso de aprendizaje busca minimizar el error (Loss) iterativamente. Como se ve en el informe, el error baja drásticamente en las primeras épocas.

Código de Implementación

entrenamiento_modelo.py 
import tensorflow as tf
import numpy as np

# 1. Datos de entrenamiento (Tensores Float32)
x_train = tf.constant([
    [50.0, 1.50], [60.0, 1.60], [70.0, 1.70], 
    [80.0, 1.80], [90.0, 1.90], [100.0, 1.70]
], dtype=tf.float32)

y_train = tf.constant([
    [22.22], [23.44], [24.22], 
    [24.69], [24.93], [34.60]
], dtype=tf.float32)

# 2. Definición de Variables (Pesos y Sesgo)
W = tf.Variable(tf.random.normal([2, 1]), name="pesos")
b = tf.Variable(tf.zeros([1]), name="sesgo")

# 3. Función del Modelo (Lineal: y = X*W + b)
def modelo_lineal(x):
    return tf.matmul(x, W) + b

# 4. Optimizador Adam
optimizador = tf.optimizers.Adam(learning_rate=0.1)

# Ciclo de entrenamiento (Simplificado)
for epoch in range(2000):
    with tf.GradientTape() as tape:
        predicciones = modelo_lineal(x_train)
        perdida = calcular_perdida(predicciones, y_train)
    
    # Calcular y aplicar gradientes
    gradientes = tape.gradient(perdida, [W, b])
    optimizador.apply_gradients(zip(gradientes, [W, b]))
